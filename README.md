# ePayco Test

Dashboard ePayco:

## Installation

1. Clone this project: `git clone <url>`.

2. Go to the project folder: `cd epayco-test`

3. Install the dependencies: `npm install`

4. Finally, you to be able to run it: `npm run dev`

## Production

You can execute the following command:

    npm run build
